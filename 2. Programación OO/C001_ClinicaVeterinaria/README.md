# Clínica veterinaria

Pequeña aplicación Java para mostrar como crear distintas clases y un programa principal que las combine.

La aplicación pretende gestionar la agenda diaria de una clínica veterinaria de una manera muy básica.

## Objetivo

- Introducción a la POO en Java. 
- Creación de clases básicas.

## Clases

- [Perro](Perro.java): representa a un perro que necesita tratamiento.
- [Veterinario](Veterinario.java): representa al veterinario que tratará a un perro.
- [Consulta](Consulta.java): representa a una cita entre un perro y un veterinario a una hora en concreto.
- [Agenda](Agenda.java): representa las citas agendadas para un día en concreto.

### Características

- Comentarios para Javadoc.
- Cada clase cuenta con un main para pruebas unitarias.
