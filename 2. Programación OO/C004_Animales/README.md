# Animales

Aplicaciones Java básicas e independientes para ilustrar los conceptos de herencia, abstracción y uso de interfaces.

## Objetivo

- Introducción a la herencia y abstracción.
- Introducción al uso de interfaces.

## Clases

- [AnimalesHerencia](AnimalesHerencia.java): ejemplo de herencia.
- [AnimalesHerenciaAbstracta](AnimalesHerenciaAbstracta.java): ejemplo de abstracción.
- [AnimalesInterfaces](AnimalesInterfaces.java): ejemplo de uso de interfaces.

### Características

- Cada clase cuenta con un main para pruebas unitarias.
- Cada fichero es independiente de los demás. 
- Uso de clases privadas estáticas para poder declarar y hacer uso de clases dentro de otras.
