# Cliente/Trabajador

Pequeña aplicación Java para mostrar el concepto de herencia.

Pretende gestionar la agenda diaria de una clínica veterinaria.

## Objetivo

- Introducción al concepto de herencia. 
- Creación de clases básicas con herencia.

## Clases

- [Persona](Persona.java): clase padre que representa a cualquier persona.
- [Cliente](Cliente.java): clase hija de Persona que representa a un cliente (especificación de persona).
- [Trabajador](Trabajador.java): clase hija de Persona que representa a un trabajador (especificación de persona).
- [Main](Main.java): programa principal.

### Características

- Comentarios para Javadoc.
- Cada clase cuenta con un main para pruebas unitarias.
