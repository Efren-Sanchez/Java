# Biblioteca 1.0

Pequeña aplicación Java que pretende gestionar el proceso de préstamos de un libro en una biblioteca.

## Objetivo

- Introducción a la POO en Java.
- Creación de clases básicas.

## Clases

- Persona: clase padre que representa a cualquier persona.
- Cliente: clase hija de Persona que representa a un cliente (especificación de persona).
- Trabajador: clase hija de Persona que representa a un trabajador (especificación de persona).
- Main: programa principal.

### Características

- Comentarios para Javadoc.
- Cada clase cuenta con un main para pruebas unitarias.
