# Programación en Java

### Programas de dificultad progresiva numerados

Colección de ejercicios de Programación en Java.
Siguen el temario de la asignatura **Programación** de 1º curso del Ciclo Formativo de Grado Superior en **Desarrollo de Aplicaciones Multiplataforma** (DAM) del sistema educativo español con una carga lectiva de 266 horas.

### Contenidos

- Programación estructurada
- Programación orientada a objetos
- Librería Collection
- Ficheros
- MySQL
- Swing
